## AI-Powered Chatbot

## Project Overview
AI-Powered Chatbot is a web-based chatbot application developed using Python and Flask. The chatbot is designed to provide automated responses to user queries using predefined FAQ data.

## Features:
* Interactive chatbot interface
* Automated FAQ responses
* User-friendly web interface
* Chat history storage using SQLite
* Fast and lightweight Flask backend
* Easy to customize with additional questions and answers

## Technologies Used
* Python
* Flask
* SQLite
* HTML
* CSS

## Installation
1. Clone the repository
```bash
git clone https://github.com/booknerdbowler008-creator/AI-Powered-Chatbot.git
```
2. Navigate to the project directory
```bash
cd AI-Powered-Chatbot
```
3. Install required packages
```bash
pip install flask
```
4. Run the application
```bash
python app.py
```
5. Open the browser and visit
```text
http://127.0.0.1:5000
```
## How It Works
1. User enters a message.
2. Flask receives the input.
3. The chatbot searches for a matching response in the FAQ database.
4. The response is displayed on the webpage.
5. User interactions are stored in the SQLite database.
